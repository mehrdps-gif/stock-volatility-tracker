# PulseInvest AI

An AI-powered investment intelligence dashboard that helps investors analyze stocks using volatility, risk metrics, market trends, and news insights.

## Features

### Stock Analysis
- Real-time stock price tracking
- Annualized volatility calculation
- Rolling volatility trends
- Risk classification

### Investment Intelligence
- AI-generated investment insights
- Dynamic pros and cons
- Buy / Hold / Watch suggestions
- Risk dashboard

### Market Intelligence
- Latest company news
- Market sentiment indicators
- Historical price analysis

### Comparison Engine
- Compare multiple stocks
- Relative volatility rankings
- Risk comparison

## Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Yahoo Finance API (yfinance)

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
